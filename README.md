## Deploy to GitHub Pages

### Method 1: Using GitHub Web Interface

1. **Create a GitHub Repository**
   - Go to [github.com](https://github.com)
   - Click "New repository"
   - Name it `username.github.io` (replace `username` with your GitHub username)
   - Make it public

2. **Upload Files**
   - Click "Add file" → "Upload files"
   - Upload `index.html`, `styles.css`, and `script.js`
   - Click "Commit changes"

3. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "GitHub Pages"
   - Source should be "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click Save

4. **Access Your Portfolio**
   - Wait a few minutes for deployment
   - Visit `https://username.github.io`

### Method 2: Using Git Command Line

```bash
# Clone/create your GitHub Pages repository
git clone https://github.com/username/username.github.io.git
cd username.github.io

# Copy portfolio files to the repository
# (Copy index.html, styles.css, script.js)

# Commit and push
git add .
git commit -m "Initial portfolio commit"
git push origin main
```

### Method 3: Using GitHub Desktop

1. Create a new repository named `username.github.io`
2. Copy the portfolio files to the repository folder
3. Commit changes ("Initial commit")
4. Push to GitHub
5. Enable GitHub Pages in repository settings
